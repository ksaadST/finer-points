Source
A Source should link to trustworthy information about the world that can be used as evidence. Useful for articles, books, studies, etc. Sources will receive votes.
Subject
A Subject is exempt from voting. Use it to link to anything that you want to reference, but not trust as direct evidence. You can analyze fiction, questionable articles, etc. without penalty.
Interpretation
Use an Interpretation to make a claim about a particular source or subject. You will probably want to include a supporting quote. Interpretations should be brief and self-evident if possible. Interpretations receive votes.
Opinion
Opinions are the place for logical analysis and broad assertions. You can draw on multiple Interpretations and/or other Opinions to make your point.
